Upload a CSV file to update inventory quantities. The file should have the following columns:

Download Template
Import Guidelines
  • Files must be in CSV format (comma-separated values)
  • First row should contain column headers
  • SKU must be unique for new products
  • For products import: existing SKUs will be skipped (not updated)
  • For inventory import: products must already exist in the system
  • Category will be matched by name (case-insensitive)
  • Decimal values should use dot (.) as decimal separator
